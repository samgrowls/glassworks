## Running Concurrent Scans

**Yes, you can run multiple scans concurrently:**

```bash
# Terminal 1: Scan category A
python3 optimized_scanner.py ai-ml-packages.txt -w 10 -e data/evidence/scan-ai -o scan-ai.json &

# Terminal 2: Scan category B
python3 optimized_scanner.py crypto-packages.txt -w 10 -e data/evidence/scan-crypto -o scan-crypto.json &

# Terminal 3: Analyze findings from scan 1
python3 batch_llm_analyzer.py flagged-ai.txt -w 3 -e data/evidence/llm-ai -o llm-ai.json &
```

**Boundaries:**
- **CPU:** 10 workers per scan (adjust with `-w`)
- **npm rate limit:** 429 errors if too aggressive (add delays if needed)
- **LLM rate limit:** 30 RPM, 60K TPM (handled by batch analyzer)
- **Disk:** ~100MB per 100 flagged packages

**Recommendation:** Run 2-3 scans concurrently max, monitor for 429 errors
